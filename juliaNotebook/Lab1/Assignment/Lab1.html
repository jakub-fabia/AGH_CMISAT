Funkcje obliczeniowe

Funkcja generalized_dot oblicza uoglniony iloczyn skalarny dwch wektorw x i y oraz macierzy A.

Funkcja matrix_vector_product oblicza iloczyn wektora x oraz macierzy A.

function generalized_dot(x::Vector, A::Matrix, y::Vector)
    return dot(x, A, y)
end

Funkcja mierzca czas obliczania

Funkcja benchmark_functions przeprowadza testy wydajnociowe dla funkcji generalized_dot oraz matrix_vector_product.
Dla rnych rozmiarw macierzy i wektorw (sizes) oraz okrelonej liczby prb (num_trials), funkcja generuje losowe macierze A oraz wektory x i y.
Nastpnie mierzy czas wykonania obu funkcji dla tych danych. Wyniki s zapisywane do pliku CSV o nazwie pomiary.csv.
Pierwsza prba dla kadego rozmiaru jest pomijana, poniewa mierzy take czas kompilacji.

In[4]:
function benchmark_functions(sizes, num_trials)
    open("pomiary.csv", "w") do file
        CSV.write(file, [], writeheader=true, header=["size", "sample", "time_dot", "time_matrix_vector"])
        for n in sizes
            for i in 0:num_trials
                
                A = rand(n, n)
                x = rand(n)
                y = rand(n)
                
                time_dot = @elapsed generalized_dot(x, A, y);
                time_matrix_vector = @elapsed matrix_vector_product(A, x);
                
                # Skip the first sample, as it measures the compilation time
                if i != 0
                    CSV.write(file, [(n, i, time_dot, time_matrix_vector)], append=true)
                end
            end
        end
    end
end
